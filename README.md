## 📦 安装方法

### 方法一：在线安装（推荐）

```bash
openclaw skill install https://github.com/charles119/chipzone-master-skill/releases/download/v1.0.0/chipzone-master.skill
```

### 方法二：下载后本地安装

1. 下载 `chipzone-master.skill` 文件
2. 在OpenClaw中运行：
   ```bash
   openclaw skill install /path/to/chipzone-master.skill
   ```
